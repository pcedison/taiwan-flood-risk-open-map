# Flood Risk API

FastAPI skeleton for public API routes and service health.

## Entry points

- FastAPI app: `app.main:app`
- Placeholder compose server: `python -m app.placeholder_server`
- Local FastAPI command after installing dependencies: `python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000`

## Current scope

- `/health` route skeleton
- Environment-based settings loader
- Structured error payload helper placeholder
- Test folder placeholder
